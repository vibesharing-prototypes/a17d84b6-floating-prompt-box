# Floating Prompt Box

## Project Overview

This is a Next.js prototype with Tailwind CSS. The main page component is in app/page.tsx.

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- TypeScript

## Development

\`\`\`bash
npm install
npm run dev
\`\`\`

## Deployment

This repo is connected to Vercel via GitHub. To deploy:

\`\`\`bash
git add .
git commit -m "Update prototype"
git push origin main
\`\`\`

Vercel auto-deploys on push. The live URL updates within ~30-60 seconds.

**IMPORTANT:** Do NOT deploy via vercel --yes, zip upload, or any API endpoint. Just git push. The Vercel project is already linked to this repo.
